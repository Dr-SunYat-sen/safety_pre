# README_先看这个

## 这个解决什么任务？

1. 查找国家安全相关调研数据；
2. 整理成表格；
3. 提供统计图或统计图方案；
4. 根据数据写出调研结论分析；

## 基本安全框架

1. 政治安全/法治教育
2. 经济安全（含粮食安全、能源安全）
3. 社会安全/公共安全
4. 网络安全/数据安全

## 文件说明

- `1_调研报告/国家安全调研数据与结论分析.md`
  - 可直接复制进总报告的分组一正文。
  - 包含调研目标、数据表、图表建议、结论分析和局限性说明。

- `2_数据表格/国家安全调研数据总表.xlsx`
  - Excel版数据总表。
  - 后续做PPT或Word表格时优先用这个。

- `2_数据表格/国家安全调研数据总表.csv`
  - CSV版数据表，便于导入其他软件。

- `3_统计图表/统计图表说明.md`
  - 说明每张图用什么数据、说明什么问题。
  - 如果PPT需要统一风格，可按这里的方案重新绘图。

- `4_数据来源/数据来源与参考文献.md`
  - 所有数据来源、发布时间、访问日期、URL、使用数据和可靠性等级。
  - 老师追问“数据从哪里来”时优先看这个文件。

- `5_过程资料/数据行备份.json`
  - 数据备份，仅做备份使用，主要用于后续修改或复核。
